# 4. 運轉操作程序

(以下變頻器參數係以 HITAKE 為例,其他變頻器請參看變頻器手冊)

| 程序 | 內容 | 確認項目 | 確認 |
|---|---|---|---|
| 1 | 電源電壓,單相電源請接於 R, T 相 | 電源電壓應與變頻器一致 | |
| 2 | NFB/操作/照明/系統/門安全 電源開關 ON 綠燈號點亮 | L49,L50,L51,L52 燈亮；顯示器顯示樓層數值或故障代碼 | |
| 3 | 安全迴路 SAR RY 指示燈亮 | CN27-A1-A4 除試俥期間外不可有短路線 | |
| 4 | 門迴路 GS+HDR RY 指示燈亮 | HDR(CN16- H1-H2)/CN27-G1-G2 除試俥期間外不可有短路線 | |
| 5 | DL & UL 指示燈亮 | H-UL(CN10-1) & H-DL(CN10-7) | |
| 6 | 確認電梯行走方向(手動模式) 變頻器參數(11-02=2) | UP時車廂應上昇,DN時車廂應下降,方向錯誤時請更改變頻器 U, V接線 | |
| 7 | 確認 Encoder 輸出 type(手動模式) | Encoder 為開集極輸出時請更改變頻器 PG-B2 卡,TA1 接線 1-3-5 短路,1=電源,2=0V,4=A 相,6=B 相 | |
| 8 | 確認 Encoder 旋轉方向(手動模式) 變頻器參數(11-02=3) | 監看變頻器輸出電流,若輸出電流異常大且電梯行走緩慢,請掉換變頻器 PG 卡 A,B 相接線 | |
| 9 | 確認 PA,PB 相序,UP 時數值增加,DN 時數值減少 | 若動作相反,PA 與 PB 對調 | |
| 10 | 調整煞車器間隙 | 煞車器動作時 BR RY 指示燈亮 | |
| 11 | 依現場規格設定 電梯運行參數 | | |
| 12 | 手動模式下檢查升降路內行程,及裝設終點開關等 | 依電梯設計規格 | |
| 13 | 試車中,若安全門履未裝設時,請先將廂上 CTBC 板 SOS1.2 兩端短接 | 主控板 OP 燈號不亮. | |
| 14 | 樓高資料設定 | 依樓高資料設定步驟進行 | |
| 15 | 電梯高速試運轉 變頻器參數設定 | 若電梯未正常減速,則樓高設定失敗,請再依設定步驟再確認 | |
