# 7.2 電梯運轉參數調整（藍綠版適用）

## 按鍵說明

| 按鍵 | 功能 |
|---|---|
| CL | "→" 欄位切換 |
| OP | "SET" 資料設定儲存 |
| HUP | "＋" 設定欄內容值＋1 |
| HDN | "－" 設定欄內容值－1 |

所有數值皆為 16 進制數值.

## 調整方法

1. 請將安全迴路切斷,控板切於手動狀態。
2. 將電源開關切斷。送電之時將 MODE 按鍵按住不放 2SEC 後將進入參數調整模式。
3. 顯示器將顯示 AA.0000,請輸入進入密碼後按 OP(ENT),密碼正確始可進入參數設定。或無密碼(A6-01,A6-00=0000 時直接進入參數設定 A0.00.XX)
4. 顯示器將顯示參數代碼及其設定值，按鍵切換至設定欄位(閃爍),按 + 或 - 鍵可改變數值至欲設定之參數數值後,再按一次鍵即進入下一欄位(閃爍位置改變)。
5. 更改資料欄位內數值,後需按 OP(ENT)鍵才可儲存數值。
6. 參數值設定完成後請將安全迴路及手自動迴路回復正常。並重新送電即脫離參數設定模式.
