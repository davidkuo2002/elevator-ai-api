# 3. 使用代碼說明（續）

| 代碼 | 說明 | 備註 |
|---|---|---|
| LAN | 外顯傳輸 | |
| NOR | 手動/自動 | NOR 與 0V 導通為自動 |
| DOR1(DO1,2) | 開門接點 1 | ON:開門 OFF:關門 |
| GS | 內門迴路繼電器 | 短路時門開後響鈴 |
| SC | 安全迴路繼電器 | |
| HDR | 外門迴路繼電器 | 短路時門開後響鈴 |
| BB | INV_BX | 可設定為反相輸出(ST) |

| 代碼 | 說明 | 備註 |
|---|---|---|
| GLS | 車廂煞車器開關 | B 接點,常態時接點導通 |
| LIGHT/FAN | 燈/風扇控制繼電器 | |
| DOR1(DO3,4) | 開門接點 2 | ON:開門 OFF:關門 |
| MHS | 救出口開關 | B 接點,常態時接點導通 |
| GKEY(GK) | 車廂管制鎖 | 常態時須與 0V 導通 |
| SOS1,2 | 門擋(門安全履) | B 接點 |
