# 6.2 故障碼（藍,綠版適用）

**格式：E X.YY.ZZ**　X=0~F 共紀錄 16 組故障紀錄,INS 時,按CL+HUP OR HDN 可調閱

YY = 變頻器運轉及 LD&LU 狀態

YY = bit7+bit6+bit5+bit4, bit3+bit2+bit1+bit0 = 16 進制碼(00H~FFH)

| Bit | 定義 | Bit | 定義 |
|---|---|---|---|
| bit0 | INV F | bit4 | INV MS3 |
| bit1 | INV R | bit5 | INV RUNNING |
| bit2 | INV MS1 | bit6 | LD 或 LU ON |
| bit3 | INV MS2 | bit7 | BK |

## ZZ 故障代碼

| 故障碼 | 發生原因 | 故障排除 |
|---|---|---|
| ZZ=00 | 無故障 | |
| ZZ=01 | THX 斷路故障 | 馬達過熱 或 變頻器回生電阻過熱。無使用時請短接。 |
| ZZ=02 | INV_RUN ERR | 變頻器無法啟動(檢查 INV_RUN 信號,變頻器配線)。 |
| ZZ=03 | DL OFF | 檢查 DL 開關。 |
| ZZ=04 | UL OFF | 檢查 UL 開關。 |
| ZZ=05 | 馬達抱閘 不 ON(電氣動作) | 請檢查 BR 接觸器動作是否正常。 |
| ZZ=B5 | 馬達抱閘不 ON(機械動作) | 請檢查主機 BR 機械煞車動作是否正常。 |
| ZZ=06 | 馬達抱閘線圈 不 OFF(電氣動作) | 請檢查 BR 接觸器接點是否黏著，此時 INVP 接觸器(B+~B-)OFF。以確保安全。 |
| ZZ=B6 | 馬達抱閘線圈不 OFF(機械動作) | 請檢查主機 BR,此時變頻器會送出"0 速度"保持車廂不滑溜。關門後回歸頂樓。 |
| ZZ=E6 | ENCODER 溜車動作偵測 | 開門中檢出 ENCODER 數值超出 A5-06 設定值，請檢查主機 BK 或 ENCODER 干擾，此時變頻器會送出"0 速度"保持車廂不滑溜。 |
| ZZ=07 | 行進中 GS OFF | 電梯行駛中，車廂門回路 OFF。請檢查門迴路。G1-G2(車廂) |
| ZZ=08 | 行進中 HDR OFF | 電梯行駛中，乘場門回路 OFF。請檢查門迴路。H1-H2(乘場) |
| ZZ=09 | 超時運轉故障 | 檢查 HOD，HOU 光電開關 或 A0-15 設定時間是否過短。 |
| ZZ=0A | 行進中 INS 回路 OFF | 請檢查手/自動迴路。 |
| ZZ=0B | INVM 不 ON | 變頻器與馬達間電磁接觸器未動作。請檢查 INVM 動作情形。 |
| ZZ=0C | ENCODER ERR | 高速運行時,ENCODER 數值未變動，請檢查 ENCODER。 |
| ZZ=12 | 行進中 INV_RUN ERR | 電梯行駛中，變頻器故障。請參閱變頻器故障碼。故障履歷 M2。 |
| ZZ=14 | 安全回路(SAR)OFF | 請檢查安全回路 A1~A4。A1-A2(車廂),A2-A3(GOV),A3-A4(升降路)。 |
| ZZ=15 / 25 | 門未關妥 | 請檢查門迴路。G1-G2(車廂),H1-H2(乘場)。 |
| ZZ=16 | 內外門都短路 | 請檢查門迴路。G1-G2(車廂),H1-H2(乘場)。 |
| ZZ=17 | 內門短路(GDR) | 請檢查門迴路。G1-G2(車廂)。 |
| ZZ=18 | 外門短路(HDR) | 請檢查門迴路。H1-H2(乘場)。 |
| ZZ=D0 | 減速距離確認 ERR | 請檢查 A5-00,01,02,03,04,05 數值 或 強減開關設定錯誤。 |

## 故障調閱

- 控制盤〈C/P〉操作開關切至 INS 位置， CL 按鍵按下時，顯示最後一次故障代碼。
- CL 鍵 + HUP/HDN 按鍵可調閱 16 次內之故障代碼,若故障代碼超過 16 次,則 EF.yy.zz 為最後一次故障紀錄。
- CL + OP 同時按鍵 5 SEC 清除所有故障代碼紀錄 (E0.yy.zz~EF.yy.zz)。
- ZZ=16,17,18 保持開門狀態並 AL 響鈴
- ZZ=06,INVP 跳脫,B+~B- 斷路,切斷煞車電源
- ZZ=B6,開門狀態下,取消所有叫車,AL 響鈴,語音及顯示警示通知,變頻器保持 0 速度直至門關妥,電梯回歸最高樓後不開門,0 速度釋放
- ZZ=E6,開門狀態下,取消所有叫車,AL 響鈴,語音及顯示警示通知,變頻器保持 0 速度直至門關妥,電梯回歸最高樓後不開門,0 速度釋放
- 要調閱故障碼時車廂之檢點操作不能切入,切入時於主機板將無法調閱故障碼。
