# 10. 線路圖 — 車廂上 CTBC 基板 I/O

> 廂上CTBC-2板 CB1接頭(CN34-2,AC11) CB1(CN34-4,AC12) 為車廂照明,風扇交流110伏電源；CB1(CN34-1,AC21) CB1(CN34-3,AC22)為門控器交流220伏電源(經由控制盤配線也可轉為交流110伏),請依設備需求電源電壓接線

## FR CN33（消防召喚扣繼電開關）

- 一次消防扣式繼電開關：FR1、0V
- 二次消防扣式繼電開關：FR2、0V

一次消防迴路開關轉動後,內叫車廂燈長按持門開到底或前往消防管制電梯啟動運行至目的樓層,如門未關閉前手動內叫門操作鈕放開,電梯門不開啟或底樓電梯不動作。

一次消防迴路開關轉動後,內叫車廂燈長按當電梯門無法關到底時,可將二次消防迴路開關轉動(需持續按壓),電梯門在全開狀態下電梯響警報電梯門動作,電梯門動作後手內叫車廂燈放開後,電梯運至目的樓層。

## MOP 車廂頂部控制器各操作開關（MOP CN40）

|---|---|
| SF1/SF2 | 緊急安全停止開關 |
| DPC/DPA | 門控電源開關 |
| 0V/24V | |
| MDC/NOR | 手動/自動開關 |
| DM/UM | 點檢下/上開關 |

## MOP CN40/CN24 MDS

- 自動：NOR
- 手動：MD_UP(UM)、MD_DN(DM)
