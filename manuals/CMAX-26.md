# 10. 線路圖 — 變頻器接線 1（15~150M/M 電梯變頻器控制端子接線，以 HITAKE 1200 為例）

## 說明

若變頻器與馬達間保護電磁接觸器不裝設時,需將 **CN14-5,13(IMS)** 兩端子短接
- CN14-5 = IML 動作輸出
- CN14-13 = IML 動作檢出

## 主控制板端 CN25（變頻器控制端）

| 0V | | |

CN25-1~11 為變頻器控制端子，11C 電纜線
CN22-2~5 = PA,PB,0V,E，連接變頻器 PG 卡 TA2，4C 隔離電纜線

## PGB_2K 端子（TA2）／ENCODER（TA1）

| TA2 | 訊號 |
|---|---|
| 1 | PA |
| 2 | 0V |
| 3 | PB |
| 4 | 0V |

TA1：白/黑/紅/綠 各 6 Pin，遮蔽網 G，push-pull 輸出 encoder／開集極輸出 encoder 二種接線方式。

## 變頻器參數對照（MS3,MS2,MS1 → 41-0x 參數）

| MS3 | MS2 | MS1 | 參數 | 說明 |
|---|---|---|---|---|
| 0 | 0 | 0 | 41-01 = | 0速度 |
| 0 | 0 | 1 | 41-02 = | 0，不使用 |
| 0 | 1 | 0 | 41-03 = | 第一段全速(60M) |
| 0 | 1 | 1 | 41-04 = | 手動速 |
| 1 | 0 | 0 | 41-05 = | 著床速 |
| 1 | 0 | 1 | 41-06 = | 救出運轉速 |
| 1 | 1 | 0 | 41-07 = | 第二段全速(90~105M) |
| 1 | 1 | 1 | 41-08 = | 第三段全速(120~150M) |

> 變頻器 JOG 端子必須設定為 MS3（T1-05=05），Y1 端子設定為（T2-02=00）
