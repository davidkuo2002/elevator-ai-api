# 2. 系統功能

| 項次 | 功能名稱 | 說明 |
|---|---|---|
| 1 | 檢修(手動)運行 | 於檢修運行模式時,按上/下行(HUP/HDN)按鈕(車廂內為 1C/2C 按紐)可使電梯以檢修速度運行(41-04),放開按鈕電梯立即停止運行.檢修運行時當電梯運行至上下強減(US/DS)開關動作時,運行速度降為著床速度(41-05). |
| 2 | 自動選擇集合運行 | 在自動狀態,電梯可以自動應答任一樓層的叫車應答,依運行方向依次應答召換. |
| 3 | 自救運行 | 當電梯處於自動狀態時,且未停止在停樓水平區時.此時只要符合可以運轉的安全狀態下,電梯將以慢速自動運行至水平停樓區後,開門. |
| 4 | 本樓開門 | 如本樓按鈕被按下,電梯門將自動打開.如按鈕按住不放,電梯門保持開啟狀態 |
| 5 | 自動開門 | 電梯應答到站水平後,電梯門自動開啟(可由參數設定,自訂開門時間) |
| 6 | 開門延長 | 當按下開門延長按鈕(HOLD)時,電梯門將自動開啟並延長開啟時間,開門延長按鈕指示燈亮(可由參數設定,自訂開門時間),需中斷開門延長時,僅需按下關門按鈕即可解除.開門延長時間到達時,BZ 鈴響通知. |
| 7 | 清除卡門異物 | 電梯當樓門因異物卡住無法順利關門時,30 sec(A0-13 預設)後自動重複開門嘗試清除異物,降低故障發生.(10 次) |
| 8 | 中文顯示電梯狀態 | LAN 傳輸板點矩陣顯示器可中文顯示目前電梯狀態 |
| 9 | 待機省電裝置 | 電梯無應答叫車超過 5 分鐘(可自定時間)車廂內照明及風扇,控制盤內變頻器自動斷電.(A0-03,A0-04) |
| 10 | 安全門邊保護 | 安全門邊採用 B 接點方式保護,不會因安全門邊斷線而無檢知開門動作,造成夾傷乘客的意外. |
| 11 | 超載保護 | 當超載開關動作時,蜂鳴器鳴響,同時電梯自動開門. |
| 12 | 樓層資料自動運算 | 採用 ENCODER 自動計算樓高及各樓層減速點，減少升降路內開關檔片及開關數量。每層樓僅需設置停車檔板一片，同時電梯車廂僅需裝設著床開關 2 個。減少試俥複雜性及降低故障率。 |
| 13 | 煞車器異常保護 | 煞車器電氣迴路與門接點串接,並設有電氣或機械偵測檢知點,確保人員安全.<br>電氣式偵測煞車器檢知點異常時,緊急中斷煞車器電源,防止煞車器異常時發生事故.<br>機械式偵測煞車器檢知點異常時,變頻器保持 0 速度力矩輸出,防止車廂不正常滑移事故.(本機能目前短路未使用，應測試確認後使用本機能) |
| 14 | UPS 運行 | 電梯運轉中停電時,若有裝設 UPS,電梯自動停靠最近樓層,不會造成緊急停車時乘客的不適及傷害. |
| 15 | 機房試運轉功能 | 可由控制盤直接叫車至任意樓層,到樓不開門. |
| 16 | 誤叫車打消功能 | 車廂有誤叫車打消功能,誤叫車時連續按鍵兩次,可打消誤叫車樓層. |
| 17 | 故障碼紀錄 | 可記錄 16 組故障紀錄,方便現場人員查修 |
| 18 | 自動回歸樓 | 可設定待機自動回歸樓層及時間(不管制).或 PAK_SW 動作後,應答所有內叫車完成後,回歸基準樓後停機.並顯示管制中.(A0-18)(A2-05)PACKING 樓與待機回歸樓為同一樓(下一次改版時建議分開以方便分別設定) |
| 19 | 門迴路短路檢出 | 自動狀態下,門全開時若「內外門」迴路都處於短路狀態,電梯保時開門狀態不運行,並 BZ 響鈴,確保安全. 單一門回路短路時,響鈴告知. |
| 20 | 滿員 PASS 功能 | 80%負載時,外叫車紀錄不應答,優先處理內叫車,需裝設 80%負載開關 |
| 21 | 卡鍵 PASS | 叫車按鍵動作超過 40SEC 時,判定為按鍵卡鍵,PASS 叫車信號 |
| 22 | 參數更改 | 電梯各項功能,輕易依現場使用環境更改參數.節省試車調試時間 |
| 23 | 車廂滑移偵測 | 車廂開門中無預期滑移時,可設定偵測滑移距離並使變頻器輸出力矩,以防止車廂移動傷人事故（A5-06） |
| 24 | 地震管制運轉 | 搭配地震感知器進行電梯管制，行進中電梯將於最近樓停車開門後停機（藍黑版） |
