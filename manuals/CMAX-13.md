# 7.1 電梯運轉參數調整（黑版適用，續）

| Address | 說明 | 預設值(HEX) | 備註 |
|---|---|---|---|
| A1-04 | LAN 33-40 站致能（40~33 樓，全部 1=全可叫車） | FF | 車廂叫車 |

範例：Bit 0 0 0 0 0 0 1 1 = 03（內叫 1~16）
對應樓層：40,39,38,37,36,35,34,33 / 32,24,16,8身障內叫到32,24,16,8樓 / 25,17,9,1一般內叫到25,17,09,1樓
