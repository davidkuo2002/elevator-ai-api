# 10. 線路圖 — 車廂內 COP 各開關說明

## 車廂操作盤 1A/2A（COP1A/2A, CN38/39）

| Pin | 功能 |
|---|---|
| RUN | 緊急安全開關 |
| NOR | 自動開關 |
| DMC | 門電源開關 |
| LIG | 電燈開關 |
| FAN | 風扇開關 |

**\*\*NOR 及 DMC 開關不使用需短接**

於 COP 內裝設 NOR 自/手動開關後,將此信號斷開車廂成手動模式便可於車廂內持續按壓 1C 或 2C 按鈕可使車廂依手動速度向下或向上行駛。

## 車廂內叫按鈕燈（CN42/43/44）

| 功能 | 端子 |
|---|---|
| 開門鈕燈 | CN42/43/44-7 OPL、CN42/43/43-9 0V、CN42/43/44-2 OP、CN42/43/44-7 24V |
| 關門鈕燈 | CN42/43/44-9 0V、CN42/43/43-9 0V、CN42/43/44-3 CL、CN42/43/44-1 24V |
| 開延長鈕燈 | CN42/43/44-7 HOL、CN42/43/43-9 0V、CN42/43/44-2 HOLD、CN42/43/44-7 24V |
| 警鈴按鈕/呼叫按鈕 | CN42/43/44-12 ALC、CN42/43/44-12 AL |
| 內叫按鈕燈 | L-、S2、S1、L+ |

## COP 內如已裝設 GK 控制開關

GK = 內叫車控制鎖,不使用此開關時 GK 需與 0V 短接
- 內叫按鈕失效開關(刷卡機開關)：CN42/43/44-11 0V、CN42/43/44-5 GK

## 車廂若需裝設叫車控制器時(刷卡機)

1. 按鈕控制 Pin3(S2)為叫車共線
2. COP板之GK開關 PIN1,PIN2為管制鎖控制,短路時,內叫車有效,不使用時需短路GK PIN1,PIN2信號源來自於CTBC頂控基CN42/CN43 CN44第5腳及第11腳

- CN10：24V, 0V, D+, D-
- DIP SW 站號=33~36
- JP1
