# 10. 線路圖 — 台達 ED 變頻器控制端子接線（15~150M/M 電梯台達ED變頻器控制端子接線）

## 說明

若變頻器與馬達間保護電磁接觸器不裝設時,需將 INVMS 兩端子短接一起
- INVM = INVM動作輸出
- INVMS = INVM動作檢出

## 主控制板端

PG：24V, 0V, PA, PB, 0V, FG（PG-1~6）
INV：INV_RUN(紫)、INV_SPD(灰)、COM(黑)、R(紅)、MS2(黃)、BB(藍) 對應 0V(白)、0V(白)、F(棕)、MS1(橙)、MS3(綠)、FG
INV-1~12 為變頻器控制端子及 PG 卡 TB2

1~10 PIN 10C電纜線；11~14 PIN 4C隔離電纜線

## 台達 ED 變頻器端子對應

| 顏色 | 主板端子 | 變頻器端子 |
|---|---|---|
| 紫 | COM | DCM |
| 黑 | F | FWD |
| 紅 | R | REV |
| 橙 | MS1 | MI1 |
| 黃 | MS2 | MI2 |
| 綠 | MS3 | MI3 |
| 藍 | BB | MI8　02-08=40 (A2-13=1) |

## 輸出接點

- R1A（02-13=12）
- R2A（02-14=29）
- R12C／RA／RB／RC（0V粉紅）

遮蔽網 G，push-pull輸出encoder／開集極輸出encoder。

## 變頻器參數

- MI8 端子設定為 (02-08=40)
- R1A 端子設定為 (02-13=12)
- R2A 端子設定為 (02-14=29)
- 04-00 = 0速度
- 04-01 = 0速度
- 04-02 = 第一段全速(60M)
- 04-03 = 手動速
- 04-04 = 著床速
- 04-05 = 救出運轉速
- 04-06 = 第二段全速(90~105M)
- 04-07 = 第三段全速(120~150M)
