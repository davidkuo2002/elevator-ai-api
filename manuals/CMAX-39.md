# 8. 主機板外觀 — OPBV-1 基板圖面（直立式）

> 本頁為 OPBV-1 操作板 PCB 版圖，含尺寸標註與端子說明，與 OPBV-1F（CMAX-38）為同系列不同外型版本。

## 板面尺寸

- 寬 45mm、高 61.6mm（整體）
- 板高標註：119mm / 125mm
- 固定孔距：40mm / 46mm，孔徑邊距 4mm

## 主要元件

- KWM-30573A0BM、KWM-30711A0BM（傳輸模組）
- CN9,CN10 = LAN 4線傳輸
- CN1~CN6（1~6 站叫車按鈕組，每組 S/L/V 端子）

## 端子功能（依站號區段）

**站號1-32：**
- CN1 = 外叫UP
- CN2 = 外叫DN
- CN3 = 殘障外叫UP
- CN4 = 殘障外叫DN
- CN5 = 上行預報燈
- CN6 = 下行預報燈

**站號33-36,41-44：**
- CN1 = 車廂叫車1
- CN2 = 車廂叫車2
- CN3 = 車廂叫車3
- CN4 = 車廂叫車4
- CN5 = 車廂叫車5
- CN6 = 車廂叫車6

**站號37-40,45-48：**
- CN1 = 車廂殘障叫車1
- CN2 = 車廂殘障叫車2
- CN3 = 車廂殘障叫車3
- CN4 = 車廂殘障叫車4
- CN5 = 車廂殘障叫車5
- CN6 = 車廂殘障叫車6

板上型號標示：OPBV-1 Ver 2.5
